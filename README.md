# India-Renewable-Hub
Here you get the details about the RE sector
A comprehensive, attractive frontend website showcasing renewable energy statistics, news, and state-wise data across India.

## Features

- **Responsive Design**: Mobile-friendly layout that works on all devices
- **State-wise Data**: Track renewable energy capacity by state (Solar, Wind, Hydro, Biomass)
- **Energy Sectors**: Overview of different renewable energy sectors
- **Latest News**: Editable news section for renewable energy updates
- **Admin Panel**: Easy-to-use interface to manage content
- **Interactive Charts**: Visual representation of energy capacity
- **Real-time Updates**: Instant updates without page reload
- **Local Storage**: Data persists across browser sessions

## Quick Start

1. Clone the repository
2. Open `index.html` in your browser
3. Navigate to the **Admin Panel** to add/edit content
